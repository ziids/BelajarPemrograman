# Belajar Pemrograman (Golang)
Jawaban Soal menggunakan Bahasa Golang

![Go](https://badgen.net/badge/icon/Golang?icon=go&label)

## Perintah Menjalankan Kode

### 1. Menghitung Total Belanja
```
go run 01-total-belanja/main.go
```

### 2. Menentukan Bilangan Genap atau Ganjil
```
go run 02-genap-ganjil/main.go
```

### 3. Menghitung Nilai Akhir Mahasiswa
```
go run 03-nilai-akhir-mahasiswa/main.go
```

### 4. Menghitung Luas Bangun Menggunakan Function
```
go run 04-luas-bangun/main.go
```

### 5. Validasi Login Sederhana
```
go run 05-login-sederhana/main.go
```

### 6. Menghitung Faktorial
```
go run 06-faktorial/main.go
```

### 7. Mencari Bilangan Terbesar dan Terkecil 3 Angka
```
go run 07-bilangan-terbesar-terkecil/main.go
```

### 8. Menghitung Jumlah Huruf Vokal
```
go run 08-hitung-vokal/main.go
```

### 9. Sistem Diskon Toko
```
go run 09-sistem-diskon/main.go
```

### 10. Sistem ATM Mini
```
go run 10-atm-mini/main.go
```
